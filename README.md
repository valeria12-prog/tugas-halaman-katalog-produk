# Tugas React - Halaman Katalog Produk

Repository tugas React untuk membuat halaman katalog produk.

## Fitur yang Berhasil Dikerjakan

✅ Menampilkan daftar produk

✅ Menggunakan React Functional Component

✅ Menggunakan React Hooks (useState)

✅ Pencarian produk

✅ Filter kategori

✅ Tampilan responsive

✅ Styling menggunakan CSS

## Cara Menjalankan

```bash
npm install
npm run dev
```

## Teknologi

- React
- Vite
- JavaScript
- CSS
